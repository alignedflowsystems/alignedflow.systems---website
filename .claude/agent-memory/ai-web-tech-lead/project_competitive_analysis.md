Competitive gap analysis completed 2026-03-28 covering the full codebase vs top agency/freelancer sites (Active Theory, Fantasy, Resn, MadeByShape, etc.).

**Why:** Business owner needs the site to win clients; analysis reveals highest-leverage gaps to close.

**How to apply:** Use these findings when scoping future features, copy improvements, and portfolio work.

## Key weaknesses identified

1. **Portfolio** — only 1 project (Monika Walek), no live URL, no client quote, no metrics/results. This is the #1 trust-killer.
2. **No pricing transparency** — services section describes packages but zero price anchoring. Prospects leave to compare competitors.
3. **No Calendly / instant booking** — only a form with 24-hour response. Top agencies offer instant scheduling.
4. **No blog / content hub** — zero organic SEO content beyond the homepage. Competitors rank for "web designer London" via articles.
5. **Testimonials not verifiable** — no links to LinkedIn profiles, Google reviews, or third-party review platforms (Clutch, Google).
6. **Stats are low-signal** — 12+ clients / 18+ projects signals "small" rather than "established". Needs reframing or removal.
7. **Hero copy is generic** — "Building for you [rotating word]" is a pattern used by hundreds of agencies built on the same template.
8. **No niche authority content** — targets wellness/coaches but nothing on the site proves deep niche expertise.
9. **No social proof beyond testimonials** — no LinkedIn, no client logos, no Clutch/Google badge.
10. **Contact page lacks urgency signals** — no average response time displayed live, no "slots available this month" signal.

## Top quick wins
- Add transparent starting prices ("from £X")
- Embed Calendly on contact page
- Add a Google Reviews badge / Trustpilot widget
- Add LinkedIn link for Mateusz in footer + team page
- Add a "Portfolio" page with richer case studies once more work exists
- Add a blog post targeting "web designer for wellness coaches London"
